# AtomicNet — brand kit

**The mark** is the product's after-state: five subsidiaries netted down to three wires.
The upper-left node is a hollow ring — Singapore nets to exactly zero, so for it, no money
moves. Never redraw the mark with more wires; the whole brand is the reduction.

## Files
- `logomark.svg` — the netted knot, on transparent. Min size 20px.
- `wordmark.svg` — mark + "AtomicNet" (Inter 700; "Atomic" paper, "Net" wire-blue).
- `favicon.svg` — mark on void, rounded square.
- `brand.css` — the token sheet (import this; don't fork values).
- `og.png` — 1200×630 social card (hero frame).

## Voice
Plain verbs, sentence case, specific over clever. The three words that carry italics in
display type: *privately*, *atomically*, *all or nothing*.

## The two rooms
- **Landing / how / proof** — "Midnight Ledger": `--void` bg, `--paper` ink, ONE accent
  (`--wire`). Currency colors appear only inside data visualizations.
- **Console** — the existing light institutional theme (`#1d4ed8` accent). The landing is
  the network at night; the console is the trading floor by day. Don't mix the rooms.

## Type
- Display: Instrument Serif (400 + italic) — headlines only, never UI.
- Body: Inter.
- Mono: IBM Plex Mono — labels, invoice refs, figures, anything that smells like a ledger.
